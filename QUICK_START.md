# 人员管理系统快速启动指南

## 🚀 立即开始

### 1. 启动应用

```bash
npm run dev
```

### 2. 初始化数据库

访问：`http://localhost:3000/init`

点击"初始化数据库"按钮，系统将自动：

- 创建数据库表
- 录入 67 名人员数据
- 设置经控贸易和开投贸易两个部门

### 3. 开始使用

- **人员管理**：访问 `http://localhost:3000/staff`
- **投票系统**：访问 `http://localhost:3000`

## 📋 功能概览

### 人员管理

- ✅ 查看所有人员（按部门分组）
- ✅ 添加新人员
- ✅ 编辑人员信息
- ✅ 删除人员
- ✅ 搜索人员
- ✅ 按部门筛选

### 预置数据

- **经控贸易**：46 名人员
- **开投贸易**：21 名人员

## 🔧 如果遇到问题

### 数据库连接问题

检查 `.env.local` 文件中的 `DATABASE_URL` 是否正确配置

### 重新初始化

如果需要重新初始化数据，再次访问 `http://localhost:3000/init`
